# Nasa Image Of The Day

## Library used
1. Spring WEB MVC
2. Java 1.8
3. Bootstrap 4.0.0


## Features
I have created a REST app with Spring WEB MVC and Bootstrap, which consumes APOD(Astronomy picture of the day) by sendind a request to NASA API.
